# Автоматическая контрастность изображения

__Usage__: `./a.out <num_of_threads> <input_image> <output_image> <coefficient>`

где _coefficient_ -- доля игнорируемого шума, диапазон значений [0, 0.5)

Диапазон значений контрастности растягивается в весь допустимый диапазон, при этом игнорируется часть самых светлых и темных точек -- это шум, который незаметен глазу, но мешает автоматической настройке контрастности. 

Программа распараллелена с помощью OpenMP.
